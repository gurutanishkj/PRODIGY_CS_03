This project is a Password Complexity Checker developed as part of the Prodigy InfoTech Cyber Security Internship (Task 03). The application is built using Python and Tkinter to provide a user-friendly graphical interface that evaluates the strength of a password in real time. It checks whether the password meets essential security requirements such as a minimum length of 8 characters, the presence of uppercase and lowercase letters, numbers, and special characters. Based on these criteria, the password is classified as Weak, Moderate, or Strong, and a progress bar visually represents its strength. If the password does not satisfy all the requirements, the application provides suggestions to help users create a more secure password. It also includes a Show/Hide Password feature for improved usability. This project demonstrates the use of regular expressions for pattern matching, GUI development with Tkinter, event-driven programming, and password security concepts, making it a practical tool for understanding password complexity and cybersecurity best practices.
